DFOnline

What is DFOnline?

Simply a service to view, share DF code online.
There is some editing and mobile support.

How do I use it?

  1. On diamondfire, break the start of a code line.
  2. While holding the item it gave you run /i nbt
  3. Open chat and click on anything sent in the message from the command. (This copies stuff onto you clipboard)
  4. Press the yellow import template button and paste your clipboard into the text field and press Import.
Once you have imported a template, you can edit it and share it with others.

How do I share templates?

While in the editing menu press File in the top left, press Export
It will open a menu, you can press Copy Short Link and share that.
Anybody who opens a shared link will be able to view and edit the template.

Import Template

Paste your template code into the text field and press Import.
