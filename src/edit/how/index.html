How to import templates to DFOnline

There are two ways to import templates to DFOnline.
The first one is can be done in normal minecraft, and the second one is done with CodeUtilities.

With boths methods, you need to look at the start of a code line, and crouch and break the block.
It will give you an item in the best slot
Hold the template, and do the one which applies to you.

Vanilla

Run /i nbt in the chat.
Then, at press the bottom big blue block of text to copy everything to the clipboard
On dfonline.dev click on the Import button.
And paste everything from the clipboard.
Press go.

CodeUtils

Open dfonline.dev
While holding the template, run /sendtemplate
CodeUtilities will have put the template data into the import menu.
Press go.

You can click on file, export and copy link to share to everyone.
