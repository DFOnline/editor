DFOnline is a service for sharing DF Code.

To import the template:

  1. In DF, break the start of the code line (event, function, ect) which will give you an item
  2. Hold this item, and type /i nbt
  3. It should send a large message, click Click to copy unformatted NBT at the bottom.
    4. This will copy the needed data onto your clipboard.
  4. On the DFOnline home page, click import template.
  5. Paste your clipboard into the text box saying Template Data and press import
    6. You will see the template and be able to edit it.

To share the template:

  1. Press File
  2. Press Export
  3. Press Copy Short Link
  4. Share the link put on your clipboard
